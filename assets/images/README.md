# Images Directory

Please place the Lorins products background image here.

**Image File Name:** `lorins-products-bg.jpg`

**Instructions:**
1. Save the Lorins product advertisement image as `lorins-products-bg.jpg`
2. Place it in this directory (`assets/images/`)
3. The login page will automatically use it as the background

**Alternative:**
If you want to use a different filename, update the `background-image` URL in `login.php`:
```css
background-image: url('assets/images/YOUR_IMAGE_NAME.jpg');
```
